## 🌟 Best FX Card (出国刷哪张卡)

**目录**: `best-fx-card/`
这是一个用于自动比价境外消费的智能工具。当你面对多种外币标价（如 $100 USD vs 15000 JPY）时，它能帮你快速找出实际人民币支付最便宜的选项，并推荐最省钱的银行卡。

### 核心功能
- **实时比价**：从快易理财网 ([kylc.com](https://www.kylc.com/huilv/whichcard.html)) 抓取各大主流银行信用卡的实时汇率。
- **全币种支持**：自动计算不同币种换算成人民币的最终支出。
- **智能排行**：直接给出省钱首选方案，并列出前 20 张最适合的信用卡排名。

### 使用方法 (Usage)

#### AI Agent 交互 (Agentic Interaction)
在 AI 对话中，你可以直接输入非结构化的自然语言，AI 会自动提取参数并调用此技能：
- **触发词示例**: "帮我算一下 100 美元和 15000 日元哪个更划算？"
- **场景指令**: "我最近要去日本旅游，有一笔 30000 日元的开销，请问用哪张信用卡最省钱？"
- **多币种对比**: "对比一下这三个价格的人民币最终支出：100 USD, 90 EUR, 15000 JPY。"

**脚本操作流程：**
1. 获取所有输入币种对应的实时汇率和各银行卡的手续费。
2. 内部汇总并对比各币种下的最便宜人民币金额。
3. 输出一份清晰的对比报表，显示哪个币种最划算，以及你应该使用哪张信用卡。

### 数据来源 (Data Source)
实时汇率与卡片费率均直接来源于 [快易理财网](https://www.kylc.com/huilv/whichcard.html)。该工具内置原生 Python 解析器，无需安装额外的重型依赖包即可运行。

## 安装方法
1. 下载或克隆本仓库。
2. 将 `best-fx-card/` 文件夹复制到你的 Agents 技能路径下。
3. 确保你的环境中安装了 Python 3。
