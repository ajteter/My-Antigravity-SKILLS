## 🌟 Best FX Card (出国刷哪张卡)

**目录**: `best-fx-card/`
这是一个用于自动比价境外消费的智能工具。当你面对多种外币标价（如 $100 USD vs 15000 JPY）时，它能帮你快速找出实际人民币支付最便宜的选项，并推荐最省钱的银行卡。

### 核心功能 (Core Features)
- **高容错输入解析**：具有极强的文本宽容度，能够自动提取并解析非标准化的输入（例如 `us$5.99`, `1500 jpy`, `46.19 SGD` 等）。
- **扩展的全币种支持**：内置常见海外币种及多种别名匹配（包含 SGD, INR, TRY, EUR, GBP, USD, JPY 等），直接比对其转化为人民币的最终实际支出。
- **弹性的网络安全机制**：默认使用标准 HTTPS 证书校验。如遇特殊的防拦截网络环境，脚本会自动安全回退并输出友好的风险提示，确保比价流程顺利走完而不崩溃。
- **机器可读模式**：支持通过附加 `--json` 参数，将比价与排序结果输出为纯 JSON 格式，方便其他自动化扩展程序或 AI 应用无缝读取与处理。
- **智能排行与实时比价**：实时从快易理财网 ([kylc.com](https://www.kylc.com/huilv/whichcard.html)) 拉取所有信用卡的汇率及结汇防坑手续费。系统不仅能锁定全局最便宜的币种方案，还会陈列出匹配该完美方案的前 20 张省钱神卡。

### 使用方法 (Usage)

#### AI Agent 交互 (Agentic Interaction)
在 AI 对话中，你可以直接输入非结构化的自然语言，AI 会自动提取多组参数并调用此技能：
- **混合批量比价**: "由于跨区标价不同，跑一下以下这组：46.19 SGD，31.00 EUR，36.26 USD，27.09 GBP，3326.83 INR"
- **触发词示例**: "帮我算一下 100 美元和 15000 日元哪个更划算？"
- **线下场景**: "去当地买一个 500 欧的东西，我手里的卡刷哪张最省钱？"

在后台，系统会自动规整成脚本命令行执行：
```bash
python3 scripts/compare.py "usd 100" "jpy 15000"
# 必要时可以附加 "--json" 参数
```

### 数据来源 (Data Source)
相关数据与汇率指标均由原生 Python 直接从 [快易理财网](https://www.kylc.com/huilv/whichcard.html) 无缝接力抓取和解析。我们在原生语法基础上搭建了完善的兼容纠错和 TLS Fallback 机制，全程零外部重型依赖。

## 安装方法
1. 下载或克隆本仓库。

3. 确保你的环境中安装了基础的 Python 3 即可顺畅运行。
